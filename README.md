# ♻️ Eco-Troca - TURMA_FSN2

## 🌱 Visão Geral

O **Eco-Troca** é uma plataforma intuitiva que transforma pequenas ações em impactos positivos, promovendo o reaproveitamento e a redução de desperdício. Além de facilitar a troca de itens, o projeto cria uma comunidade colaborativa comprometida com a sustentabilidade e a responsabilidade ambiental.

---

## 🎯 Objetivos do Projeto

- 🔄 **Facilitar trocas** entre usuários para reaproveitamento de itens
- 🌍 **Reduzir o impacto ambiental** via reutilização de bens
- 💚 **Criar uma comunidade** engajada em sustentabilidade
- 👩‍💻 **Oferecer interface intuitiva** e acessível

---

## 🛠️ Funcionalidades

- **Cadastro e Login:** Sistema seguro de autenticação
- **Listagem de Itens:** Cadastre e anuncie itens para venda ou troca
- **Busca e Filtros:** Encontre itens por categoria e cidade
- **Sistema de Mensagens:** Comunicação direta entre usuários
- **Perfil de Usuário:** Seus itens e trocas

## 🚀 Instalação

1. **Clone o repositório**
    ```
    git clone https://github.com/mateusHtech/Eco-troca-TURMA_FSN2.git
    cd Eco-troca-TURMA_FSN2
    ```

2. **Instale as dependências**
    ```
    npm install
    ```


3. **Execute a aplicação**
    ```
    npm run dev
    ```

4. **Acesse no navegador:**  
    [http://localhost:5173/](http://localhost:5173/)

## 👩‍🔧 Uso

1. **Criar uma Conta:** Cadastre-se usando e-mail válido.
2. **Anunciar Itens:** Adicione itens para troca com fotos e descrição.
3. **Explorar Itens:** Use filtros ou barra de pesquisa para buscas fáceis.
4. **Negociar Trocas:** Use o chat ou envie através do botão para combinar uma troca.

## 🤝 Contribuição

Quer colaborar? Siga o fluxo:

1. Faça um **fork** do repositório.
2. Crie uma branch para sua alteração:
    ```
    git checkout -b nome da sua branch
    ```
3. Faça **commit** das alterações:
    ```
    git commit -m 'Descrição da sua alteração'
    ```
4. **Push** para o repositório remoto:
    ```
    git push origin nome da sua branch
    ```
5. **Abra um Pull Request** descrevendo suas mudanças.
